How to use this bot:
1) go to main.py and change user_username and user_pwd to your creditentials.
2) download chromedriver from https://chromedriver.chromium.org/ and choose the right version for your own chrome browser version
3) if you have loading problems, you may want to change the n in sleep(n) to whatever number you need at what moment you need it (these sleep functions are in all of the files, you need to understand how the code works to know which ones to change)

Run the main.py file with python 3 and follow the instructions
